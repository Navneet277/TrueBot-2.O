About TrueBot

TrueBot is a fake news detection project that combines modern NLP preprocessing, TF-IDF features, and ensemble machine learning to provide clear fake/real predictions with confidence scores.

Tech Stack

  • Flask + Bootstrap responsive UI
  • TF-IDF vectorization with Logistic Regression / Random Forest
  • Curated synthetic dataset covering popular rumor themes
  • SQLite for history tracking
